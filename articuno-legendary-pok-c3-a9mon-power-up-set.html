Articuno (Japanese: フリーザー Freezer) is a Special Pokémon Ga-Olé disk. It is one of the Promotional disks.

Release information

This disk was available as part of the Legendary Pokémon Power Up Set ~Articuno, Zapdos & Moltres Ver.~, released on April 25, 2019. The set also includes a Zapdos disk, a Moltres disk, and a standard Ga-Olé Pass.
